How to Play
♟
A board position appears after a few moves. Guess the chess opening. Each wrong guess reveals one more move.
6
You have 6 guesses. Start with 1-3 moves shown depending on difficulty.
📊
Each guess reveals 5 attribute scores - Family, Move 1, Structure, ECO, and Tier.
Family
Same opening family? (e.g. Sicilian)
Move 1
Does White's 1st move match?
Structure
Similar pawn setup? (open/closed)
ECO
How close is the ECO code?
Tier
Same difficulty tier? (Beginner/Inter/Master)
Green - exact or very close match
Yellow - partially correct / same category
Red - wrong
